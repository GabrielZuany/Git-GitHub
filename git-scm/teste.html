#GIT E GITHUB Guia prático de comandos no git bash. git init: inicia um repositório. git add: adiciona um repositorio ou arquivo(para arquivos contidos em pastas: git add pasta/arquivo); git commit -m "" : faz o commit; git log: exibe as alterações; git status: exibe o estado do desenvolvimento; git show: mostra o último ponto da história; git show + numero do commit: exibe commit especifico; git branch nome/nome: cria uma ramificação; git checkout nomebranch: muda para a branch criada;
